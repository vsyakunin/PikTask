# Микросервис с API для проверок станций метро.
Тестовое задание в компанию ПИК.ПРО для позиции Junior Python Developer.
Выполнил Всеволод Якунин.

Данный микросервис написан на Python 3.7.5.

## Установка зависимостей
Для установки зависимостей выполните:
```bash
pip install -r requirements.txt
```

## Запуск приложения
Для запуска на локальной машине выполните:
```
set FLASK_APP=webapp && set FLASK_ENV=development && set FLASK_DEBUG=1 && flask run
```
